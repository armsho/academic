There seems to be a demand for a polished, fully functional template that would be useful to professors, researchers, and students. I designed and developed this template with a focus on simplicity, customizability, and ease of upkeep. The design effectively displays one’s publications, current and completed projects, teaching, and various professional accomplishments in a clean and organized manner. I anticipate that this template provides a solid starting point for establishing a visible presence on the web for academic purposes. Feel free to help this template evolve and have access to the code on my GitHub repository.
